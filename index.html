Политика конфиденциальности

Обновлено: 20 августа 2025

1. Какие данные мы собираем

  • Telegram ID пользователя
  • Имя пользователя (если доступно)
  • Список каналов, которые вы подключаете
  • Статистика использования (дата активации, время активности, платежи и т.п.)

2. Как мы используем данные

  • Для предоставления доступа к функциям бота
  • Для расчёта статистики и аналитики
  • Для администрирования подписки и платежей
  • Для соблюдения законодательства (в случае необходимости)

3. Хранение и защита данных

  • Данные хранятся в защищённой базе данных
  • Доступ к данным ограничен техническим персоналом
  • Все платёжные данные обрабатываются через сторонние платёжные системы (мы не храним карты)

4. Передача данных третьим лицам

Мы не передаём ваши персональные данные третьим лицам без вашего согласия

5. Удаление данных

По запросу пользователя все его данные могут быть удалены (через команду /delete или поддержку)

6. Контакты

По вопросам конфиденциальности: @BotFlowSupport

Правила поведения пользователей

Не нарушайте правила Telegram:

Запрещено распространять запрещённый контент (экстремизм, порнография, спам и т.д.)

Честное использование:

  • Не используйте бота для мошенничества или накрутки
  • Один пользователь = один аккаунт. Запрещено обходить систему подписок

Уважайте поддержку:

За грубость, угрозы или попытки взлома — блокировка

Важное примечание:

Все действия пользователя фиксируются в системе для предотвращения нарушений
